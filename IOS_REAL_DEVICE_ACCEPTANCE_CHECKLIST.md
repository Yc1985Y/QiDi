# iOS 真机验收清单

更新时间：`2026-07-02`

本清单用于 TestFlight 安装后的 iPhone/iPad 真实验收。所有功能必须是真实实现，不允许 mock/demo/假数据/placeholder/空壳按钮冒充完成。

## 基础启动

- [ ] iPhone 可安装 TestFlight 版本。
- [ ] iPhone 冷启动成功。
- [ ] iPhone 热启动成功。
- [ ] iPad 可安装 TestFlight 版本。
- [ ] iPad 冷启动成功。
- [ ] iPad 热启动成功。
- [ ] Display Name 显示为 `织时`。
- [ ] App 图标显示正常。
- [ ] 启动画面显示正常。

## 页面与核心输入

- [ ] 首页显示正常，没有明显遮挡、错位或无法点击。
- [ ] 文本输入可提交真实内容。
- [ ] 文本输入会进入真实事项解析流程。
- [ ] 相册输入可选择真实图片。
- [ ] 相册图片会进入真实 OCR/解析流程。
- [ ] 拍照输入可调用真实相机。
- [ ] 拍照图片会进入真实 OCR/解析流程。
- [ ] 系统分享入口可从外部 App 唤起《织时》。
- [ ] 系统分享文本可进入真实解析流程。
- [ ] 系统分享图片可进入真实解析流程。

## 事项与保存

- [ ] 事项解析返回真实结果。
- [ ] 解析失败时显示真实失败原因。
- [ ] 确认卡片展示标题、时间、地点、说明、置信度。
- [ ] 确认卡片可以完成确认操作。
- [ ] 确认后保存到时间线。
- [ ] 时间线显示已保存事项。
- [ ] 关闭并重启 App 后，时间线数据仍存在。

## 系统能力

- [ ] 系统日历可被调起。
- [ ] 确认后的事项可写入系统日历。
- [ ] 本地通知权限可申请。
- [ ] 本地通知可按真实时间触发。
- [ ] PDF 可真实导出。
- [ ] PDF 可打开。
- [ ] PDF 可通过系统分享面板分享。

## 异常与权限

- [ ] 拒绝相机权限后有真实错误提示。
- [ ] 拒绝相册权限后有真实错误提示。
- [ ] 拒绝通知权限后有真实状态提示。
- [ ] 拒绝日历权限后有真实错误提示。
- [ ] 无网络时有真实错误提示。
- [ ] vivo API 失败时有真实错误提示。
- [ ] 没有 mock/demo 模式入口。
- [ ] 没有假数据默认填充真实结果。
- [ ] 没有 placeholder 页面冒充功能完成。
- [ ] 没有空壳按钮。
